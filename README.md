# 🐡 Pokemon 
A full stack pokemon app built with Redux & React 

## Features 
- 🐡 Add new pokemon to the database 
- 💦 Update pokemon attack & defense moves 
- 🍭 Add & Update Items that can be given to the Pokemon

## Running Locally 
```
cd backend 
npm install 
npm start

cd frontend
npm install
npm start
```
